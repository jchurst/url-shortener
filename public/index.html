URL Shortener Microservice

*Best viewed with Firefox (JSON rendering)

Enter a parameter in the format below to receive a response:

Pass a URL as a parameter to receive a shortened URL as a JSON response:

Example invalid URL:
http://www.flame-chord.glitch.me/new/foo.com

Example creation usage:
http://www.flame-chord.glitch.me/new/http://www.google.com

Example creation output:
{"original_url":"http://www.google.com", "short_url":"http://www.flame-chord.glitch.me/r1piS7ybz"}

Usage:
http://www.flame-chord.glitch.me/r1piS7ybz

Will redirect to:
http://www.google.com
